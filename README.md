# Search Engine Rank Automation

This project automates search engine ranking checks using Selenium.

## Features
- Check domain ranking on Google
- Check ranking on Bing
- Check ranking on DuckDuckGo
- Export results to CSV

## Tech Stack
- Python
- Selenium
- WebDriver Manager

## Installation

Clone repository

git clone https://github.com/gaurav91489/testbookrankautomation.git

Install dependencies

pip install selenium webdriver-manager

## Run Script

python rank_checker.py

## Output

The script generates CSV files containing ranking positions for each search phrase.
